### 指针数组
声明：
```c
char *buff[];
```
**`C`的声明规则就是：从标识符开始，按运算符优先级和结合方向**\
按 `C` 的声明规则，从 `buff` 开始读：

1. `buff` 是一个数组
2. `[]` 里的每个元素
3. 类型是 `char *`


假设：
`char *buff[] = { "abc", "def", "ghi" };`

内存结构是：

```text
buff:
+-------+-------+-------+
|  ptr  |  ptr  |  ptr  |
+-------+-------+-------+
   |       |       |
   v       v       v
 "abc"   "def"   "ghi"
```

- `buff` 自身：指针数组
- 每个元素：指向一个字符串首地址

### `sizeof` & `strlen`

| 对比点         | `sizeof`         | `strlen`           |
| ----------- | ---------------- | ------------------ |
| 本质          | 运算符              | 函数                 |
| 发生时机        | **编译期**          | **运行期**            |
| 计算对象        | 类型 / 对象的**内存大小** | C 字符串的**长度**       |
| 是否关心 `'\0'` | 不关心              | **以 `'\0'` 为结束标志** |
| 是否遍历内存      | ❌                | ✅（逐字节扫）            |
